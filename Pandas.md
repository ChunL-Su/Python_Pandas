# Pandas
- 1、一维数据表Series基础使用
```
import pandas as pd
import numpy as np

s = pd.Series([1,3,6,np.nan,44,1])
print(s)
>>>
0     1.0
1     3.0
2     6.0
3     NaN
4    44.0
5     1.0
dtype: float64
```
